# Daily Tasks

Projeto desenvolvido com foco na prática de JavaScript, manipulação de DOM e organização de interface.

## 📎 Live Demo
👉 https://biancassp.github.io/daily-tasks/

## 📌 Sobre o projeto
A proposta é uma lista de tarefas simples, onde é possível adicionar atividades do dia a dia de forma rápida e organizada. 

Interações da aplicação:
- Ao clicar **uma vez**, é possível editar o texto da tarefa, facilitando correções ou alterações;
- Ao clicar **duas vezes**, a tarefa é marcada como concluída;
- Também é possível **excluir tarefas**, se necessário.

Além disso, o projeto conta com uma mensagem motivacional que muda automaticamente de acordo com o dia, incentivando o usuário a se manter focado e cuidar da própria rotina.
